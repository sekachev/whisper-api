# Whisper API

This is a simple API for OpenAI's Whisper speech-to-text model, built with FastAPI.

## Installation

1. Clone the repository:
git clone https://github.com/sekachev/whisper-api.git cd whisper-api


2. Install dependencies:
pip install -r requirements.txt



3. Run the API:
python main.py



## Features

- Transcribes audio files using OpenAI's Whisper.
- Supports batch processing.
- Provides a system tray icon for starting/stopping the server.
